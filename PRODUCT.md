# Product

## Register

product

## Users

War Era game players and community journalists who need to convert entity names (countries, regions, alliances, parties, military units) into in-game URLs for weekly reports and news articles.

## Product Purpose

Resolve plain-text entity names into War Era game URLs using naive string matching + AI-powered filtering. The tool saves time for reporters who manually write weekly war reports and need to link every mentioned entity.

## Brand Personality

Dark, sharp, utilitarian. A gaming tool that feels like a terminal — fast, no fluff, focused.

## Anti-references

Generic SaaS landing pages, cream/beige backgrounds, gradient text, glassmorphism, over-rounded cards.

## Design Principles

1. Speed over decoration — every element earns its place
2. Dark mode by default — gaming context demands it
3. Minimal chrome — the textareas are the product
4. Functional color — red for actions, muted for everything else

## Accessibility & Inclusion

WCAG AA contrast targets. Reduced motion support. Keyboard-accessible (Ctrl+Enter shortcut). Monospace output for precise text editing.
